# Test Recordings

Place `.mp3`, `.wav`, or `.m4a` meeting recordings here for Deepgram accuracy testing.

## Requirements

- Minimum 3 recordings
- Recommended: 5+ minutes each
- Multi-speaker meetings preferred (for diarization testing)
- Real meeting audio (not synthetic) for meaningful accuracy evaluation

## Privacy Note

These files are for local testing only. Do NOT commit recordings to git.
Add `recordings/*.mp3`, `recordings/*.wav`, `recordings/*.m4a` to `.gitignore`
if not already covered.
